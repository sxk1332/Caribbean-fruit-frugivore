Author: Seokmin Kim
Project: Frugivores explain plant dispersal syndrome diversity in the Caribbean

This repository contains the supplemental materials, data, and scripts used for the project.
See "Docs" folder for the supplemental materials mentioned within the manuscript text.
See "Data" folder for data used. These are divided into abiotic, biotic, island biogeography, and plant folders.
Data folder had to be zipped. 

Within the Data folder, the abiotic materials can be obtained from online sources as mentioned within the readme_abiotic text file (within the Data folder).

See "Scripts" folder for R codes used for all analyses.
